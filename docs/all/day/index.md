Each game session is played over a number of rounds called days. Each day is divided into the five phases:

- [New Day Phase](/docs/all/day/new-day-phase)
- [Overland Phase](/docs/all/day/overland-phase)
- [Encounter Phase](/docs/all/day/encounter-phase)
- [Reward Phase](/docs/all/day/reward-phase)
- [End of Day Phase](/docs/all/day/end-of-day-phase)
